> # OpenDB - 数据库运维工具设计文档
>
> ## 1. 项目概述
>
> ### 1.1 项目背景
> OpenDB 是一款基于 Tauri v2 (Rust) + React 19 技术栈开发的现代化数据库运维工具，旨在提供与 Navicat 相当的功能体验，同时增强 AI 辅助能力、多语言支持和主题切换功能。本工具将支持主流数据库系统，特别强调对 GaussDB 25.1.32 和 openGauss 最新版本的原生驱动支持。
>
> ### 1.2 核心目标
> - **功能完整性**：实现 Navicat 的核心功能，包括数据库连接、数据建模、SQL 编辑、数据同步等
> - **兼容性**：支持高版本和低版本数据库，特别支持 GaussDB 25.1.32 和 openGauss 的原生驱动
> - **AI 能力**：集成 AI 助手，提供智能 SQL 生成、错误诊断、性能优化建议
> - **用户体验**：简洁直观的界面设计，支持主题切换和中英文多语言
> - **性能与安全**：利用 Rust 语言的高性能和内存安全特性，确保工具的稳定性和安全性
>
> ### 1.3 项目范围
> **MVP (最小可行产品) 包含**：
> - 连接管理（多数据库类型，含 GaussDB）
> - SQL 编辑器（语法高亮、自动补全、执行）
> - 数据查看与编辑
> - Schema 浏览器
> - 数据导入/导出（CSV、SQL 脚本、JSON）
> - AI 助手（连接外部 AI，自然语言转 SQL、建表、查询）
> - 离线安装支持（独立客户端，无需网络）
>
> **后续迭代扩展**：
> - AI 深度集成（SQL 优化建议、慢查询分析、智能 Schema 设计）
> - 数据传输与同步
> - 团队协作功能
> - NoSQL 支持（MongoDB、Redis）
> - Web 版（基于同一套前端代码）
>

> ### 3.2 GaussDB 驱动架构设计
>
> #### 3.2.1 驱动关系确认
> - **openGauss**：开源关系型数据库，基于 PostgreSQL 内核，采用木兰宽松许可证 v2 发行
> - **GaussDB**：华为基于 openGauss 内核开发的企业级分布式数据库产品
> - **兼容性**：GaussDB(for openGauss) 与 openGauss 保持高度兼容性
> - **驱动策略**：GaussDB 和 openGauss 使用独立的驱动模块，确保最佳兼容性和性能
>
> #### 3.2.2 驱动模块设计
> - **模块化架构**：每个数据库类型使用独立的驱动模块
> - **GaussDB 25.1.32 驱动**：
>   - 使用纯 Rust 实现的 `tokio-gaussdb` 驱动
>   - 通过动态加载机制，避免 Java 依赖
>   - 支持高可用和分布式特性
> - **openGauss 驱动**：
>   - 基于 PostgreSQL 协议的优化版本
>   - 支持 openGauss 特有语法和功能
>   - 兼容社区版 openGauss
> - **统一接口层**：
>   - 提供一致的 API 调用接口
>   - 隐藏底层实现细节
>   - 支持动态切换驱动
>
> #### 3.2.3 驱动管理机制
> - **动态加载**：按需加载对应数据库驱动
> - **自动更新**：驱动自动下载、验证和更新
> - **版本兼容**：支持多个版本的数据库，自动适配最佳驱动版本
> - **安全验证**：驱动文件 SHA256 校验，确保完整性
>
> ## 4. 功能模块设计
>
> ### 4.1 数据库连接管理
> - **多数据库支持**：GaussDB 25.1.32、openGauss、PostgreSQL、MySQL、SQLite、MSSQL、ClickHouse
> - **连接配置**：
>   - 支持 SSL/TLS 加密连接
>   - 支持 SSH 隧道连接
>   - 支持代理连接
> - **连接池管理**：智能连接池，自动回收闲置连接
> - **凭据安全**：使用 OS Keyring 存储凭证，AES-256 加密本地配置
> - **连接测试**：一键测试连接可用性
> - **驱动管理**：独立驱动模块，最佳兼容性
>
> ### 4.2 SQL 编辑与执行
> - **语法高亮**：支持多种数据库方言，智能识别当前连接类型
> - **代码补全**：表名、字段名、函数、关键字自动补全
> - **执行计划**：可视化执行计划展示，性能瓶颈高亮
> - **历史记录**：自动保存查询历史，支持搜索和重用
> - **格式化**：SQL 代码自动格式化，符合编码规范
> - **多协议支持**：针对 GaussDB 和 openGauss 的特定协议优化
>
> ### 4.3 数据管理
> - **数据浏览**：表格化数据展示，支持分页、排序、筛选
> - **数据编辑**：
>   - 行内编辑，支持文本、数字、日期、布尔类型
>   - 新增/删除行
>   - BLOB 字段特殊处理
>   - 外键导航
> - **数据导入**：支持 CSV、Excel、JSON、SQL 文件导入
> - **数据导出**：多种格式导出（CSV、Excel、JSON、SQL），支持大文件分块
> - **数据统计**：自动统计行数、空值率、数据分布
> - **事务管理**：支持事务回滚、提交，确保数据一致性
>
> ### 4.4 数据建模与设计
> - **ER 图设计**：可视化 ER 图设计，自动识别外键关系
> - **表设计器**：字段管理、约束管理、索引管理、触发器管理
> - **正向工程**：从设计生成 DDL 语句
> - **逆向工程**：从数据库生成 ER 图和表结构
> - **版本对比**：不同版本 schema 对比，变更可视化
> - **跨数据库兼容**：支持不同数据库类型的 DDL 转换
>
> ### 4.5 AI 助手功能
> - **设计理念**：AI 助手采用侧边栏对话模式，不打扰主工作流
> - **上下文感知**：AI 自动获取当前连接的 Schema 信息作为上下文
> - **多模型支持**：支持多种 AI 服务
>   - OpenAI (GPT-4o / GPT-4)
>   - Azure OpenAI
>   - Ollama (本地)
>   - 通义千问 (Qwen)
>   - DeepSeek
>   - 自定义 API
> - **核心功能**：
>   - **自然语言转 SQL**：用自然语言描述需求，AI 生成 SQL 查询语句
>   - **AI 建表**：通过自然语言描述表结构，AI 生成 CREATE TABLE
>   - **查询分析**：AI 解释查询结果含义，分析数据趋势
>   - **SQL 优化建议**：AI 分析查询性能瓶颈并给出优化建议
>   - **慢查询分析**：AI 分析执行计划，识别潜在性能问题
>   - **智能错误修复**：AI 分析 SQL 错误并给出修复建议
>   - **智能数据建模**：AI 根据业务需求自动设计表结构和索引
> - **GaussDB 适配**：
>   - AI 自动识别 GaussDB 并适配其特有语法
>   - 支持 GaussDB 特有的函数：sysdate(), nvl(), decode() 等 Oracle 兼容函数
>   - 支持 GaussDB 特有的分区表语法：PARTITION BY RANGE/LIST/HASH
>
> ### 4.6 安全设计
> - **Tauri Isolation 模式**：iframe 沙箱隔离前端，所有 IPC 消息经加密通道传输
> - **认证与授权**：
>   - 桌面端：OS 用户级隔离
>   - Web 端：JWT + RBAC
> - **凭证安全**：
>   - OS Keyring 存储 (macOS Keychain / Windows Vault)
>   - AES-256-GCM 加密 (本地配置文件)
>   - 零明文策略 (前端从不持有密码明文)
> - **传输安全**：
>   - TLS 1.2+ 数据库连接加密
>   - SSH 隧道 (远程连接)
>   - HTTPS (Web 端 API)
> - **输入安全**：
>   - SQL 参数化查询 (防注入)
>   - CSP 内容安全策略
>   - 输入验证与清理
>
> ## 5. 兼容性与部署
>
> ### 5.1 支持平台
> - **Windows**：Windows 10/11 (64位)，支持 ARM64
> - **macOS**：macOS 14+ (Apple Silicon & Intel)
> - **Linux**：Ubuntu 22.04+, CentOS 8+, Debian 12+
>
> ### 5.2 部署方式
> - **安装包**：平台原生安装包 (.msi, .dmg, .deb, .rpm)
> - **便携版**：绿色版，无需安装，单文件执行
> - **企业部署**：支持静默安装、组策略配置
> - **更新机制**：自动更新检查，增量更新支持
>
> ### 5.3 系统要求
> - **最低配置**：
>   - CPU: 双核 2.0 GHz
>   - RAM: 4 GB
>   - 存储: 500 MB
>   - 显示: 1280x720
> - **推荐配置**：
>   - CPU: 四核 3.0 GHz
>   - RAM: 8 GB
>   - 存储: 1 GB SSD
>   - 显示: 1920x1080
>
> ### 5.4 离线安装方案
> - **设计原则**：完全离线安装，零依赖运行，单文件分发
> - **GaussDB 驱动内嵌方案**：
>   - 使用纯 Rust 实现的 `tokio-gaussdb` 驱动
>   - 编译后直接内嵌到二进制文件中
>   - 无需 Java 运行时，无需手动下载 JDBC 驱动
>   - 支持 openGauss 和华为 GaussDB
> - **离线安装包内容**：
>   - Windows (.msi)：~18MB，包含 WebView2 Bootstrapper
>   - macOS (.dmg)：~15MB，开箱即用
>   - Linux (.AppImage)：~20MB，包含所有依赖库
